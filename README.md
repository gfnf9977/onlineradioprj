Вимоги до системи:
.NET SDK 9.0.
Будь-яка IDE (Visual Studio 2022, VS Code, Rider).

Порядок запуску:
Розпакуйте архів із проєктом/створити нову папку, запустити термінал та вписати git clone https://github.com/gfnf9977/onlineradioprj# .
Переконайтеся, що у папці RadioStationSolution.WebApp/wwwroot/ffmpeg/ наявні файли ffmpeg.exe та ffprobe.exe (вони необхідні для роботи аудіо-конвертера).
Запустіть проєкт:
у терміналі перейти до папки, де перебуває файл рішення RadioStationSolution.sln : cd RadioStationSolution.WebApp та ввести dotnet run
Проєкт налаштовано на використання SQLite. Встановлення SQL Server не потрібне. При першому запуску система автоматично cтворить файл бази даних RadioStation.db у папці проєкту, створить таблиці та наповнить їх відповідними даними. Оскільки SQLite вже запускалась, дані вже там присутні, дані облікових записів зареєстрованих користувачів можна переглянути в файлі accounts.txt, що так само є в папці проєкту. Порада: Для перевірки функціоналу сповіщень рекомендується зареєструвати нового користувача, використавши реальну електронну пошту. Система надішле вітальний лист, що підтвердить роботу SMTP-сервісу.
